Function: utility/product-codes

Link: https://api.intrapost.nl/utility/product-codes

Description

Collect available product codes for customer

Request:

* = required

Supported content types:
application/json: example
application/xml: example

  • ApiKey*: API key that you can find on the documentation page
  • AccountNumber*: customer accountnumber to be used

Response:

Supported accept types:
application/json: example
application/xml: example

  • Errors: list of error messages when an error has occured, otherwise null
  • ProductCodes: the list of available product codes
    • ProductCode: productcode
    • ParcelTypeNameNld: parceltype (dutch)
    • ParcelTypeNameEng: parceltype (english)
