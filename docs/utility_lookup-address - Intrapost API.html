Function: utility/lookup-address

Link: https://api.intrapost.nl/utility/lookup-address

Description

Address lookup via zipcode

Request:

* = required

Supported content types:
application/json: example
application/xml: example

  • ApiKey*: API key that you can find on the documentation page
  • Number*: house/P.O. box number (has to be in range between 1 and 999999)
  • Zipcode*: zipcode

Response:

Supported accept types:
application/json: example
application/xml: example

  • Errors: list of error messages when an error has occured, otherwise null
  • Street: street
  • City: city
