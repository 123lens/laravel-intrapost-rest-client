Function: utility/dropoffpoints-for-international-address

Link: https://api.intrapost.nl/utility/dropoffpoints-for-international-address

Description

lookup of addresses of dropoff points via international address

Request:

* = required

Supported content types:
application/json: example
application/xml: example

  • ApiKey*: API key that you can find on the documentation page
  • AccountNumber*: customer accountnumber to be used
  • Limit: maximum number of results
  • Street: street
  • Number*: house number (has to be in range between 1 and 999999)
  • Zipcode*: zipcode
  • Countrycode*: countrycode

Response:

Supported accept types:
application/json: example
application/xml: example

  • CarrierType: carrier
    Supported values:
    • [1] PostNL: PostNL
    • [2] GLS: GLS
    • [4] DHL: DHL
  • PickUpPoints: list of nearby pickup points
    • CarrierId: Unique id of this pickup point (has to be at most 100 characters long)
    • Name: company name (has to be at most 150 characters long)
    • Street: street (has to be at most 150 characters long)
    • Number: house/P.O. box number (has to be in range between 1 and 999999)
    • Addition: housenumber addition (has to be at most 15 characters long)
    • Zipcode: zipcode (has to be at most 15 characters long)
    • City: city (has to be at most 100 characters long)
    • CountryCode: countrycode (has to be at most 2 characters long)
    • Distance: distance in meters to point of origin
    • GeoLocation: exact geographical location
      • Latitude: latitude of location
      • Longitude: longitude of location
    • OpeningHours: opening hours for this location
      • Day: day of the week
      • Time: opening hours
