Function: track-trace/get-from-vz

Link: https://api.intrapost.nl/track-trace/get-from-vz

Description

Get parcels by their 'find-my-shipment' code. Maximum of 50 pieces per request

Request:

* = required

Supported content types:
application/json: example
application/xml: example

  • ApiKey*: API key that you can find on the documentation page
  • VZCodes*: list of Intrapost VZCodes
  • IncludeHistory: defines whether the status history will be needed or not

Response:

Supported accept types:
application/json: example
application/xml: example

  • Errors: list of error messages when an error has occured, otherwise null
  • Shipments: the list of found shipments
    • ID: the shipment's unique ID
    • VZCode: the 'vind mijn zending' code
    • ParcelType: product type
      Supported values:
      • [1] StandardParcel: a standard parcel
      • [2] InsuredParcel: an insured parcel
      • [3] RegisteredParcel: a registered parcel
      • [4] InsuredLetter: an insured letter
      • [5] RegisteredLetter: a registered letter
      • [6] StandardParcelStatedAddress: a standard parcel delivered only on stated address
      • [7] StandardParcelStatedAddressSignature: a standard parcel delivered only on stated address with signature
      • [8] MailboxParcel: mailbox parcel
      • [9] ParcelViaPickupLocation: a standard parcel delivered on pickup location
      • [10] StandardParcelWithEveningDelivery: a standard parcel with evening delivery
      • [11] StandardParcelWithAgeCheck: a standard parcel with age check
      • [12] StandardParcelSignature: a standard parcel delivered with signature for receipt
    • Address: address
      • Street: street (has to be at most 150 characters long)
      • Number: house/P.O. box number (has to be in range between 1 and 999999)
      • Addition: housenumber addition (has to be at most 15 characters long)
      • Zipcode: zipcode (has to be at most 15 characters long)
      • City: city (has to be at most 100 characters long)
      • CountryCode: 2-letter ISO countrycode (has to be 2 characters long)
      • Company: company name (has to be at most 150 characters long)
      • FullName: full name (has to be at most 150 characters long)
      • Email: email address (has to be at most 150 characters long)
      • Phone: phonenumber (has to be at most 20 characters long)
      • CountryNameNL: country name in Dutch
      • CountryNameEN: country name in English
    • AccountNumber: account number that the item is declared under
    • TrackTraceLink: track & trace link
    • ExternalLink: external link
    • Barcode: external barcode
    • Reference: reference
    • WeightInGrams: weight in grams
    • WeightInKilograms: weight in KG
    • Status: the current status
      Supported values:
      • [0] Created: created/pre-registered
      • [1] Received: received for delivery
      • [2] Sorted: sorted for delivery
      • [3] Transported: on its way for delivery
      • [4] Delivered: delivered
      • [5] Returning: is being returned
      • [6] Returned: is returned
      • [7] Rerouted: delivery address is modified
      • [8] Pickup: can be collected at pickup point
      • [9] DeliveryDateTimeChanged: delivery date/time is modified
      • [10] PickupCollected: is collected at pickup point
      • [11] Processed: processed by Intrapost
      • [13] DeliveryFailed: delivery failed
    • StatusTimestamp: the current status
    • StatusDescriptionNL: status description (in Dutch)
    • StatusDescriptionEN: status description (in English)
    • DeliveryFrom: start of delivery time window
    • DeliveryTill: end of delivery time window
    • PickupAddressLines: pickup address
    • StatusHistory: the list of status history
      • Status: the status classification
        Supported values:
        • [0] Created: created/pre-registered
        • [1] Received: received for delivery
        • [2] Sorted: sorted for delivery
        • [3] Transported: on its way for delivery
        • [4] Delivered: delivered
        • [5] Returning: is being returned
        • [6] Returned: is returned
        • [7] Rerouted: delivery address is modified
        • [8] Pickup: can be collected at pickup point
        • [9] DeliveryDateTimeChanged: delivery date/time is modified
        • [10] PickupCollected: is collected at pickup point
        • [11] Processed: processed by Intrapost
        • [13] DeliveryFailed: delivery failed
      • StatusTimestamp: the date and time for this status
      • StatusDescriptionNL: status description (in Dutch)
      • StatusDescriptionEN: status description (in English)
    • Specials: the list of status history
      • OptionType: the type of the special
        Supported values:
        • [1] WasLetter: Was letter
        • [2] DeviatedProcessing: Deviated Processing
        • [3] DimensionSurcharge: Dimension Surcharge
        • [4] IslandSurcharge: Island Surcharge
        • [9] IsLarge: Is Large
        • [11] WasMailBoxParcel: Was mailbox parcel
        • [12] SpecialSize: Special Size
        • [13] Fragile: Fragile
      • OptionDescription: description of the special
    • Measurements: the list of status history
      • OptionType: the type of measurement
        Supported values:
        • [0] IntrapostLength: Intrapost measurement of Length
        • [1] IntrapostWidth: Intrapost measurement of Width
        • [2] IntrapostHeight: Intrapost measurement of Height
        • [3] CarrierLength: Carrier measurement of Length
        • [4] CarrierWidth: Carrier measurement of Width
        • [5] CarrierHeight: Carrier measurement of Height
      • OptionDescription: the type of measurement
      • OptionValue: measurement in centimeters
