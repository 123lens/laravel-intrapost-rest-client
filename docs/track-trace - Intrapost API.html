track-trace API

Create, manage, and retrieve track & trace shipments

Functions:

  • create: Register a shipment
  • create_2_0: Register a shipment (v2.0)
  • create_3_0: Register a shipment (v3.0)
  • create_mailboxparcel: Register a postalbox parcel for transporter labelin ZPL format
  • get-retour-label: Create return label for existing shipment
  • cancel: Cancel a shipment
  • search: Search for shipments
  • get-from-id: Get parcels by their ID. Maximum of 50 pieces per request
  • get-from-vz: Get parcels by their 'find-my-shipment' code. Maximum of 50 pieces per request
  • create-labels: Retrieve shipping labels in one file
