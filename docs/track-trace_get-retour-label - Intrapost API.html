Function: track-trace/get-retour-label

Link: https://api.intrapost.nl/track-trace/get-retour-label

Description

Create return label for existing shipment

Request:

* = required

Supported content types:
application/json: example
application/xml: example

  • ApiKey*: API key that you can find on the documentation page
  • AccountNumber*: customer accountnumber to be used
  • Product: product, we currently only support standard parcel
    Supported values:
    • [1] StandardParcel: a standard parcel
    • [2] InsuredParcel: an insured parcel
    • [3] RegisteredParcel: a registered parcel
    • [4] InsuredLetter: an insured letter
    • [5] RegisteredLetter: a registered letter
    • [6] StandardParcelStatedAddress: a standard parcel delivered only on stated address
    • [7] StandardParcelStatedAddressSignature: a standard parcel delivered only on stated address with signature
    • [8] MailboxParcel: mailbox parcel
    • [9] ParcelViaPickupLocation: a standard parcel delivered on pickup location
    • [10] StandardParcelWithEveningDelivery: a standard parcel with evening delivery
    • [11] StandardParcelWithAgeCheck: a standard parcel with age check
    • [12] StandardParcelSignature: a standard parcel delivered with signature for receipt
  • ShipmentID: the unique ID of the shipment
  • VZCode: the 'find my shipment' code

Response:

Supported accept types:
application/json: example
application/xml: example

  • Errors: list of error messages when an error has occured, otherwise null
  • TrackTraceLink: track & trace link
  • Barcode: the barcode from PostNL
  • PdfLabelData: label PDF file in base64 encoding
