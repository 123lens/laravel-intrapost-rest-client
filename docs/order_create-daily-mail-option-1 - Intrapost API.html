Function: order/create-daily-mail-option-1

Link: https://api.intrapost.nl/order/create-daily-mail-option-1

Description

create a daily mail order with option 1 (no amounts specified)

Request:

* = required

Supported content types:
application/json: example
application/xml: example

  • ApiKey*: API key that you can find on the documentation page
  • AccountNumber*: customer accountnumber to be used
  • DeliveryTime*: the delivery time of the pieces to Dutch addresses
    Supported values:
    • [0] Standard: standard
    • [3] FixedDays: fixed days
  • CrateInfo: Information about the postcrates
    • Amount: amount of cates
    • Description: description and/or unique characteristics of the crates
  • Reference: reference, personal input (has to be at most 2048 characters long)

Response:

Supported accept types:
application/pdf: example
application/json: example
application/xml: example

  • Errors: list of error messages when an error has occured, otherwise null
  • FileName: the name the PDF file has in our systeem
  • OrderCode: the order's code, which is unique per order
  • Base64PdfContent: the content of the PDF in base64 encoding
